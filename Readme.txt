Use Unix, as I have developed and run code in Unix.
IDE used : Atom
Environment Required : Unix.

To Run application.
1. Unzip the folder.
2. In console, traverse to application folder.
3. Install Beautifulsoup
4. Run the app.py

The server is up and running. Now, go to browser and URL window type localhost:5000. On the page you might see
that the latitude and longitude are defaulted with Arlington's coordinates.
You should put latitude in first text box and longitude in second text box.

Known issues :
1. The callouts are made as per guided in https://graphical.weather.gov/xml/. But there are times
when all the reuested weather parameters are not fetched from the server.
